# HealthGo UI

HealthGo is a healthcare-focused frontend web platform prototype designed to improve accessibility and simplify user interaction through a clean and user-friendly interface.

## Features

* Clean healthcare-focused UI
* Accessibility-oriented design
* User-friendly navigation flow
* Frontend prototype structure

## Tech Stack

* HTML
* CSS
* JavaScript
* UI/UX Planning
* Canva

## Purpose

This project was created as a frontend healthcare platform concept to explore digital accessibility, interface planning, and user-focused web experiences.
